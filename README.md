# SilverCare-Innovating-Healthcare-Acces-for-the-Golden-Age-
It is an application where elderly people can write their symptoms, and the app provides possible diagnoses they might have. The ‘Emergency’ button, when pressed once, calls the family, and when pressed twice, calls the ambulance. At the same time, it also tells them what to do in case of an emergency.
SilverCare is a healthcare application designed especially for elderly people.
The app helps users monitor their health by entering symptoms and receiving possible diagnoses based on their condition.

Features

Symptom checker for elderly users
Possible diagnosis suggestions
Emergency assistance button
Voice assistance and guidance
Simple and accessible interface
How It Works

The user enters their symptoms into the app.
The application analyzes the symptoms.
The app displays possible diagnoses that the user might have.
Voice support can read important information aloud.
Emergency Button

The app includes a special Emergency button:

Press once → calls a family member
Press twice → calls an ambulance
The application also provides instructions about what to do in emergency situations while help is arriving.

Technologies Used

Swift
SwiftUI
Xcode
Purpose

SilverCare was created to help elderly people get quick medical guidance and emergency support in an easy and accessible way.
